# BuyVistas - Premium Home Decor

A complete static eCommerce website for home decor products.

## How to Use

Simply open `index.html` in your web browser to view the website.

## Structure

```
├── index.html          # Home page (START HERE)
├── shop.html           # All products
├── product.html        # Product details
├── cart.html           # Shopping cart
├── checkout.html       # Checkout form
├── about.html          # About us
├── contact.html        # Contact page
├── privacy-policy.html
├── terms.html
├── refund-policy.html
├── shipping-policy.html
├── css/
│   └── styles.css      # All styles
├── js/
│   ├── products.js     # 25 products data
│   ├── cart.js         # Cart functionality
│   └── main.js         # General scripts
└── assets/             # Images folder
```

## Features

- 25 Premium home decor products
- 6 Categories: Wall Art, Lighting, Decor, Furniture, Kitchen, Bedroom
- Shopping cart with localStorage persistence
- Responsive design for all devices
- No server required - works offline!
